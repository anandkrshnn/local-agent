## 🚀 60-second Launch

**Docker (Recommended)**:
```bash
curl -sSL https://raw.githubusercontent.com/anandkrshnn/local-agent/main/docker-run.sh | bash
# http://localhost:8000
```

**Docker Compose**:
```bash
git clone https://github.com/anandkrshnn/local-agent
cd local-agent && mkdir vaults
docker-compose up -d
```
